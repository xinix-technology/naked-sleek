Icons

Naked CSS is Use Font Awesome for Icon.

You can place Font Awesome icons just about anywhere using the CSS Prefix xn and the icon's name. Font Awesome is designed to be used with inline elements (we like the <i> tag for brevity, but using a <span> is more semantically correct).
